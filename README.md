# 🍰 Сайт кондитерской «Зёрнышко»

Современный, эстетичный веб-сайт для локальной кондитерской-пекарни «Зёрнышко». Проект разработан с упором на аппетитный визуал, удобное интерактивное меню и адаптивность под мобильные устройства.

---

## 🎯 Цель проекта
Создание коммерческого сайта для презентации ассортимента кондитерской, привлечения локальных клиентов и автоматизации приема предзаказов на торты и десерты.

## ✨ Планируемый функционал и фичи
* **🧁 Интерактивное витринное меню:** Удобное разделение десертов по категориям (Кофе, Десерты, Закуски) с ценами и составом.
* **📅 Форма заказа/предзаказа:** Интерактивная форма, где клиент может выбрать дату, время и оставить заявку на индивидуальный торт.
* **📸 Галерея изделий:** Сетка с качественными фотографиями готовой продукции и элементами плавного увеличения при наведении.
* **📍 Блок контактов и карта:** Интеграция интерактивной карты для удобного поиска кондитерской.

## 🛠️ Стек технологий
* **HTML5:** Семантическая верстка, формы ввода данных, оптимизированная структура для SEO.
* **CSS3:** Flexbox/Grid для построения кондитерской витрины, кастомные анимации появления элементов, адаптивные медиа-запросы.

## 💻 Локальный запуск
1. Клонируйте репозиторий.
2. Откройте файл `index.html` в браузере.
